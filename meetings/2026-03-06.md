# 4 Discussions

## 4.1 Runtime Keys

**Ethan Arrowood** incorporated Aki’s first round of feedback for the technical report that will represent runtime keys. There are just a few more edits to be made from Ecma’s feedback, and Ethan is seeking more input from the committee.

Ethan will be ready to present a final draft to the committee in a future meeting.

Aki suggested the committee consider utilising multiple source control aaS services to host the ongoing drafts of the TR in addition to GitHub.

## 4.2 CLI API

**Ishat Gupta** suggested a new global called “Environment” a class resembling

```typescript
class Environment {
  variables: Record<string, string>
  channels: Record<string, Stream>
}
```

This would require a [Permissions API](https://developer.mozilla.org/en-US/docs/Web/API/Permissions_API), which still has more work to be done from the web platform before it’s ready for TC55 to tackle. The chairs requested that Ishat put together a small proposal on the agenda for delegates to review before the next meeting.
